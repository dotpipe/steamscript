# js_shell App Catalog

This document lists and describes all available apps in the `apps/` directory of js_shell. Each app is a standalone JavaScript file that can be run from the shell prompt.

| App Name      | Description                                                      |
|-------------- |------------------------------------------------------------------|
| alias         | Manage command aliases                                           |
| apropos       | Search for commands by keyword                                   |
| basename      | Print the last part of a file path                               |
| basenameall   | Print the last part of multiple file paths                       |
| basenameext   | Print the base name of a file without its extension              |
| bc            | Simple calculator (evaluates expressions)                        |
| build         | Build utility (custom, see source)                               |
| cal           | Print a calendar for a given month and year                      |
| cat           | Print file contents                                              |
| chmod         | Change file permissions (simulated)                              |
| chown         | Change file owner (simulated)                                    |
| clear         | Clear the shell screen                                           |
| cp            | Copy files                                                      |
| curl          | Download files from URLs                                         |
| date          | Print the current date and time                                  |
| dateadd       | Add days to a date                                               |
| datemath      | Perform date arithmetic                                          |
| dc            | Desk calculator (expression evaluator)                           |
| df            | Show disk usage (simulated)                                      |
| dotpipe       | Launch DotPipe remote connection window                          |
| du            | Show directory usage (simulated)                                 |
| echo          | Print arguments to the screen                                    |
| env           | Show or set environment variables                                |
| exit          | Exit the shell                                                   |
| explorer      | Launch file explorer (if supported)                              |
| export        | Export environment variables                                     |
| expr          | Evaluate expressions                                             |
| factor        | Factor a number                                                  |
| find          | Find files by name                                               |
| findall       | Find all files matching a pattern                                |
| ftp           | Simple FTP client (simulated)                                    |
| grep          | Search for text in files                                         |
| head          | Print the first lines of a file                                  |
| help          | Show help for commands                                           |
| history       | Show command history                                             |
| index         | Index files (custom, see source)                                 |
| kill          | Kill a process (simulated)                                       |
| less          | View file contents page by page                                  |
| ls            | List files in a directory                                        |
| lsdir         | List directories                                                 |
| lsfile        | List files only                                                  |
| man           | Show manual for a command                                        |
| mkdir         | Create a directory                                               |
| more          | View file contents page by page                                  |
| mv            | Move or rename files                                             |
| nano          | Simple text editor                                               |
| ping          | Ping a host (simulated)                                          |
| ps            | Show process status (simulated)                                  |
| pwd           | Print working directory                                          |
| resetkey      | Reset your shell key                                             |
| rm            | Remove a file                                                    |
| rmdir         | Remove a directory                                               |
| scp           | Secure copy (simulated)                                          |
| seq           | Print a sequence of numbers                                      |
| set           | Set shell variables                                              |
| sleep         | Pause for a number of seconds                                    |
| sort          | Sort lines in a file                                             |
| ssh           | SSH client (simulated)                                           |
| tail          | Print the last lines of a file                                   |
| tar           | Archive files (simulated)                                        |
| thisApp       | Demo app for shell scripting                                     |
| touch         | Create an empty file                                             |
| tree          | Show directory tree                                              |
| type          | Show the type of a command                                       |
| uniq          | Filter adjacent matching lines                                   |
| units         | Convert units (simulated)                                        |
| unset         | Unset shell variables                                            |
| unzip         | Extract zip files (simulated)                                    |
| usermgr       | Manage users (admin only)                                        |
| vi            | Text editor (vi-like)                                            |
| wc            | Word, line, and byte count                                       |
| wget          | Download files from the web                                      |
| which         | Show the path of a command                                       |
| whoami        | Show current user                                                |
| zip           | Create zip archives (simulated)                                  |

For more details on each app, see the source code in the `apps/` directory.
